# VibroBalance 📊

Приложение для балансировки роторов методом влияния коэффициентов.

## ✨ Возможности

- 🔧 **Балансировка**: расчёт корректирующего груза, векторная диаграмма
- 📊 **История**: сохранение всех балансировок с поиском
- 💾 **Коэффициенты**: библиотека коэффициентов влияния
- 📄 **PDF-отчёты**: генерация профессиональных отчётов
- 📥 **Экспорт/Импорт**: сохранение и загрузка данных в JSON

## 📥 Установка

Скачайте последнюю версию из [Releases](https://github.com/shonentez-afk/VibroBalance-apk/releases)

## 🛠 Требования

- Android 7.0+ (API 24)
- 50 MB свободного места

## 📖 Как использовать

1. Выполните балансировку (вкладка "Балансировка")
2. Просмотрите историю (вкладка "История")
3. Сгенерируйте PDF-отчёт (откройте запись → "Сохранить PDF")
